## Getting Started

### Install
```bash
yarn install
# or
npm install
```

### To run in development mode:

```bash
yarn dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

[API Response Example](https://run.mocky.io/v3/32409c32-c694-40e6-a118-f0f888f7e355)

## Deploy
### Build
```bash
yarn build
# or
npm run build
```

### Start
```bash
yarn start
# or
npm run start
```
